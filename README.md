# Mega Weapons

A Fabric Minecraft mod focused on high-speed combat, weapon abilities, combo mechanics, and anime-inspired movement.

Currently adds:
- Sword upgrade system
- Dash ability
- Combo-based Black Flash attacks
- Axe paralysis attacks
- Token economy progression system

Built for Fabric on Minecraft 1.21.1.

---

# Features

## Dash Ability
- Press the custom Dash keybind while sprinting with a sword
- Dash forward with high-speed movement
- Configurable in Minecraft Controls menu

## Sword Leveling
- Sneak + Right Click with a sword to upgrade it
- Costs 1 Mega Token (Gold Nugget)
- Higher levels increase dash power

## Black Flash Combo System
- Land consecutive sword hits
- Every 4th hit triggers:
  - Massive knockback
  - Sonic Boom particles
  - Special effects

## Axe Paralysis
- Sneak + attack with an axe
- Applies extreme slowness to targets

## Mega Tokens
- Killing mobs grants Mega Tokens
- Tokens are currently represented by Gold Nuggets
- Used for upgrading weapons

---

# Requirements

- Minecraft 1.21.1
- Fabric Loader
- Fabric API

---

# Installation

1. Install Fabric Loader
2. Install Fabric API
3. Place the mod `.jar` into your mods folder

---

# Controls

| Action | Default Key |
|---|---|
| Dash | R |

The Dash key can be changed in:
`Options → Controls → Mega Weapons`
